# Lab Airflow

Pipeline desenvolvido para a disciplina de Orquestração de Workflows utilizando Apache Airflow.

## Funcionalidades

- ETL com TaskFlow API
- TaskGroup
- Dynamic Task Mapping
- Callbacks de sucesso e falha
- Retry com backoff exponencial
- Integração com FakeStore API
- Persistência em PostgreSQL utilizando PostgresHook
- UPSERT utilizando ON CONFLICT
- Pool para controle de concorrência

## Tecnologias

- Apache Airflow 2.9
- Docker Compose
- PostgreSQL
- Python
